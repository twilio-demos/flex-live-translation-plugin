# ConversationRelay Translator Live Translation Plugin

This plugin works in correspondence with the ConversationRelay (CR) Translator application. Read [here](https://github.com/twilio-demos/ConversationRelay-Translator/) for more info.

## Usage Guide

This plugin automatically connects with the CR Translator once you have configured your profile and set the worker handle.

When the specified worker picks up a call, it will show a tab with "Live Translation" which will poll the application and display the original and translated text for both the caller and worker.

![live translation](/docs/images/live-translation.png)

## Deploy This Yourself?

### Have Also Deployed CR Translator

If you are using the already deployed CR Translator app, you can **skip this step**.

In `use-conversation.tsx` change:

```tsx
const response = await fetch(
  `https://smn0iajild.execute-api.us-east-1.amazonaws.com/active-conversation?workerHandle=${workerHandle}`
);
```

To

```tsx
const response = await fetch(
  `https://<your_url>/active-conversation?workerHandle=${workerHandle}`
);
```

### Release The Plugin

Read this [guide](https://www.twilio.com/docs/flex/quickstart/getting-started-plugin) if you do not have the Twilio CLI or Flex setup.

```bash
twilio flex:plugins:deploy --major --changelog "style translation" --description "CR Translator Plugin"
twilio flex:plugins:release --plugin plugin-sample@2.0.0 --name "Autogenerated Release 1770828219843" --description "The description of this Flex Plugin Configuration."
```
